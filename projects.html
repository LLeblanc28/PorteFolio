Projets

E-Inscription

Projet de deuxième années de BTS, cette application permettait aux futurs étudiants de s'inscrire dans leurs formations. Cette application a été développée en PHP avec le framework Symfony, Une base de données Mysql a été utilisée pour stocker les données des étudiants et des formations. L'application a été conçue afin de limité notemment l'utilisation de papier et de faciliter le travail des secrétaires et des étudiants. Cette application recensait des formulaires d'inscriptions, étaient envoyer directement depuis Parcoursup.

Guitar Configurator

Guitar Configurator est une application Front only utilisant une API, elle permet à l'utilisateur de configurer sa guitare en choisissant les différentes pièces qui la composent, comme le corps, le manche, les micros, etc. L'application a été développée en Angular et utilise une API pour récupérer les données sur différents type de guitares. L'application a été conçue afin de permettre aux utilisateurs de visualiser leur configuration de guitare en temps réel et de comparer deux configurations.

AutoFact

AutoFact est une application de facturation pour les entrepreneurs. Elle permet ainsi au entrepreneur d'avoir un logiciels open-source pour gérer leur facturation, leurs clients et leurs produits. L'application a été développée en C# avec le framework .NET, une base de données SQL Server a été utilisée pour stocker les données des clients, des produits et des factures. L'application a été conçue afin de permettre aux entrepreneurs de gérer facilement leur facturation et de suivre l'état de leurs factures en temps réel.

SAFM

SAFM est une application de gestion de flotte de véhicules pour les entreprises. Elle permet aux entreprises de gérer leur flotte de véhicules, de suivre l'entretien et de planifier les réparations. L'application a été développée en PHP avec le framework Symfony et utilise une base de données MySQL pour stocker les données des véhicules, des entretiens et des réparations. L'application a été conçue afin de permettre aux entreprises de gérer efficacement leur flotte de véhicules
